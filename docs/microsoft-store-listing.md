## Description
*(Up to 10 000 characters. Use line breaks between paragraphs.)*

```
Gitember is a free, open-source Git GUI client that turns the complexity of Git into a clean, visual experience — no terminal knowledge required.

Whether you just started using Git or you're managing a large team repository, Gitember gives you a full-featured environment for everything from daily commits to deep code archaeology.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 NEW IN VERSION 3 — LIGHTER & FASTER
━━━━━━━━━━━━━━━━━━━━━━━━━━

Version 3 replaces the heavyweight JavaFX runtime with a lightweight Swing UI. The result: faster startup, a smaller install footprint, lower memory usage — and a snappier experience on every machine. No bloated runtime, no slow splash screen.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 FULL GIT SUPPORT
━━━━━━━━━━━━━━━━━━━━━━━━━━

Clone, init, commit, push, pull, fetch, branch, merge, rebase, stash, cherry-pick, tag — every essential Git operation is available through an intuitive point-and-click interface. Works with SSH and HTTPS remotes.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 POWERFUL FILE DIFF VIEWER
━━━━━━━━━━━━━━━━━━━━━━━━━━

Three diff modes in one tool:
• Unified diff — colour-coded additions and deletions in a single pane
• Context diff — surrounding lines keep the changes in perspective
• Side-by-side diff — dual-pane view for effortless line-by-line comparison

Syntax highlighting covers 40+ languages including Java, Python, JavaScript, TypeScript, Kotlin, Go, Rust, C/C++, SQL, YAML, XML, Markdown, and more.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 FOLDER COMPARISON TOOL
━━━━━━━━━━━━━━━━━━━━━━━━━━

Compare any two directory trees side by side — build outputs, backup copies, feature branches. Files are colour-coded as Added (green), Removed (red), or Modified (blue). Double-click any changed file to open the full diff viewer. Configurable ignore lists automatically skip compiled artifacts (.class, .dll, .pyc, and more).

━━━━━━━━━━━━━━━━━━━━━━━━━━
 PULL REQUEST REVIEW — RIGHT INSIDE THE APP
━━━━━━━━━━━━━━━━━━━━━━━━━━

Connect Gitember to your hosting platform and review Pull Requests without leaving the app:
• GitHub (github.com and GitHub Enterprise)
• GitLab (gitlab.com and self-hosted)
• Bitbucket Cloud
• Gitea (self-hosted)

See open PRs, browse changed files colour-coded by status, and diff any file between base and head branch with a double-click. One-click jump to the PR page in your browser.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 ADVANCED FULL-TEXT SEARCH
━━━━━━━━━━━━━━━━━━━━━━━━━━

Gitember's indexing engine searches across your entire repository history — not just source code. Find content inside:
• Source code, Markdown, YAML, and config files
• Microsoft Office documents (Word, Excel, PowerPoint)
• OpenDocument files (Writer, Calc, Impress)
• PDFs
• Images (PNG, JPEG, SVG — indexed metadata and OCR-extracted text)
• CAD files (DXF, STL)

Filter results by branch, author, or date range.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 REPOSITORY STATISTICS
━━━━━━━━━━━━━━━━━━━━━━━━━━

Understand project health at a glance. Visual reports show commit frequency, lines changed per author, branch activity, and average branch lifetime. Perfect for sprint reviews, code audits, and team retrospectives.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 GIT LFS — LARGE FILE STORAGE
━━━━━━━━━━━━━━━━━━━━━━━━━━

Built-in Git LFS support for teams that manage binary assets — images, videos, 3D models, datasets, and design files. Track LFS patterns, fetch objects, and see LFS file status directly in the UI.

━━━━━━━━━━━━━━━━━━━━━━━━━━
 MORE FEATURES
━━━━━━━━━━━━━━━━━━━━━━━━━━

• Visual commit graph with branch and merge lines
• File-level history — trace every change to a specific file
• One-click diff between any two commits
• Restore any file to any historical version in two clicks
• Git blame / change annotation — see who changed every line and when
• Stash management — save, browse, and apply stashes
• Mass operations — stage, unstage, revert, delete multiple files at once
• Ed25519 SSH key support
• Works offline — no account or internet connection required
• Open source — Apache License 2.0, source on GitHub
```

## Privacy Policy Summary

Gitember collects **no personal data** and makes **no network requests** on its own except:
- Cloning / fetching from Git remotes that the user explicitly configures.
- Fetching Pull Request data from GitHub / GitLab / Bitbucket / Gitea using a personal access token entered by the user.

No telemetry, no analytics, no account required.

Full privacy policy: https://gitember.app/privacy
